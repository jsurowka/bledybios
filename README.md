# Bledybios

Prosta aplikacja internetowa w PHP na potrzeby projektu

## Co to robi?
- wyświetla stronę w przeglądarce,
- obsługuje proste błędy (np. przez plik `beep-errors.php`),
- może być używana jako baza pod inne ćwiczenia.

## Jak uruchomić?
1. Postaw lokalny serwer (np. WAMP na Windowsie).
2. Wrzuc katalog z projektem do folderu serwera, np. `c:\wamp64\www\bledybios`.
3. Włącz serwer.
4. Wejdź w przeglądarce na adres:
   - `http://localhost/bledybios`
